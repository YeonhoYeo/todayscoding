안녕하세요, 매일프로그래밍 이번주 문제입니다.
 
      길이가 같은 두 문자열(string) A 와 B가 주어지면, A 가 B 로 1:1 암호화 가능한지 찾으시오.
      
      Given two strings of equal length, check if two strings can be encrypted 1 to 1.
      
      
      예제)
      
      Input: “EGG”, “FOO”
      
      Output: True // E->F, G->O
      
      
      Input: “ABBCD”, “APPLE”
      
      Output: True // A->A, B->P, C->L, D->E
      
      
      Input: “AAB”, “FOO”
      
      Output: False
