      String이 주어지면, 중복된 char가 없는 가장 긴 서브스트링 (substring)의 길이를 찾으시오. 
      
      Given a string, find the longest substring that does not have duplicate characters.
      
      예제)
      
      Input: “aabcbcbc”
      
      Output: 3 // “abc”
      
      
      Input: “aaaaaaaa”
      
      Output: 1 // “a”
      
      
      Input: “abbbcedd”
      
      Output: 4 // “bced”
